# Storyboards - Flujos de Usuario

## Descripción
Aquí documentamos los *User Journeys* diseñados para **BookEat**. En estos *storyboards* se ilustra la lógica de 
navegación entre las páginas, resultado de la interacción de los usuarios con la aplicación.

> Estos diseños permiten asegurar una experiencia de usuario fluida incluso antes de pasar a la maquetación final.

---
